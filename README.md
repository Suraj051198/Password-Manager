# 🔐 Password Manager

A full-stack **Password Manager** web application built using the **MERN** stack and styled with **Tailwind CSS**. It allows users to securely store, manage, and retrieve passwords with an intuitive UI and backend security features.

---

## 🚀 Tech Stack

**Frontend:**
- ⚛️ React.js
- 💨 Tailwind CSS
- 🔐 Axios for API calls

**Backend:**
- 🟩 Node.js
- 🚂 Express.js

**Database:**
- 🍃 MongoDB (Mongoose ODM)

**Others:**
- 📦 dotenv for environment config
- 🔒 bcrypt for password hashing
- 🛡️ JWT for authentication
- 🧪 Postman for API testing

---

## 💡 Features

- 🔐 User Authentication (JWT)
- 📂 Secure storage of passwords
- 📋 Copy password to clipboard
- 🧠 Responsive UI with Tailwind CSS
- 🔄 RESTful APIs for CRUD operations
- 🧪 Backend tested with Postman
